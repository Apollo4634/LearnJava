# 容器命令 volume

`docker volume`包含了若干关于数据卷操作的子命令：

|         命令          | 说明                             |
| :-------------------: | -------------------------------- |
| docker volume create  | 创建数据卷                       |
| docker volume inspect | 查看一个或者多个数据卷的具体信息 |
|   docker volume ls    | 列举数据卷                       |
|  docker volume prune  | 移除所有未使用的本地数据卷       |
|   docker volume rm    | 移除一个或者多个数据卷           |
