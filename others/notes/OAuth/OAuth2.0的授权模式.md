# OAuth2.0的授权模式

OAuth2.0提供了4种授权模式：**隐式授权模式**（Implicit Grant）、**授权码授权模式**（Authorization code Grant）、**密码模式**（Resource Owner Password Credentials Grant）、**客户端凭证模式**（Client Credentials Grant）。
